# Cofind 2 UX Audit

## Scope

Проверен интерфейс по DOM/CSS и пользовательским потокам:

- Главная: hero, live-заявки, live-чат, последние просмотры.
- Лента: фильтры, сортировка, карточки, пагинация.
- Заявка: detail, отклик, автор, похожие заявки.
- Создание заявки: форма, WYSIWYG, preview, каталог.
- ЛК: профиль, роль кабинета, готовность, мои заявки, уведомления, блок-лист.
- Чат и inbox: комнаты, поиск, история, composer, рисунки.
- Админка: role-aware секции, таблицы, каталоги, финансы/SEO/аудит.
- Auth/static pages: вход, регистрация, восстановление, help/rules/privacy/contacts.

## Fixed In This Pass

- Добавлен системный `focus-visible` для клавиатурной навигации по ссылкам, кнопкам, полям, rich editor и tabindex-элементам.
- Добавлен `overflow-wrap: anywhere` для заголовков, карточек, rich-контента, сообщений и служебных подсказок, чтобы длинные URL/slug/email не ломали сетку.
- Поля ввода получили более явное состояние фокуса.
- Кнопки получили минимальную высоту, а на мобильных - более комфортные touch-targets.
- На мобильных кнопочные группы и заголовочные действия лучше растягиваются по ширине.
- Убрано масштабирование `font-size` через viewport units; типографика теперь управляется фиксированными брейкпоинтами.
- Добавлен `pnpm ux:audit`, который статически проверяет UX-контракты: focus-visible, mobile breakpoints, перенос длинного текста, image optimization, отсутствие viewport-scaled font-size и non-zero letter-spacing.
- Ранее в этом же блоке были добавлены responsive-точки 720px/560px, горизонтальная прокрутка admin-таблиц, ограничение высоты чата и превью рисунков.
- Изображения avatar/cover/background/drawing оптимизируются на клиенте перед загрузкой.
